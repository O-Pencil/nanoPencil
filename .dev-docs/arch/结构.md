## дјҳеҠҝеҜ№жҜ”

### вқҢ ж—§з»“жһ„ (еҚ•дёҖд»“еә“)

```
nanoPencil/
в”ңв”Җв”Җ core/extensions/nanomem/  (еөҢе…Ҙеӯҗзӣ®еҪ•)
в”ңв”Җв”Җ core/
в”ңв”Җв”Җ modes/
в””в”Җв”Җ ...

й—®йўҳпјҡ
вҖў nanomem ж— жі•зӢ¬з«ӢдҪҝз”Ё
вҖў ж— жі•еҚ•зӢ¬зүҲжң¬з®ЎзҗҶ
вҖў жөӢиҜ•е’Ңжһ„е»әиҖҰеҗҲ
```

### вң… ж–°з»“жһ„ (Monorepo)

```
packages/
в”ңв”Җв”Җ nanomem/        (зӢ¬з«ӢеҢ…)
в”ңв”Җв”Җ nanopencil/     (дё»еҢ…)
в””в”Җв”Җ simplify/       (еҸҜйҖүжү©еұ•)

дјҳеҠҝпјҡ
вң“ nanomem еҸҜзӢ¬з«ӢеҸ‘еёғе’ҢдҪҝз”Ё
вң“ жё…жҷ°зҡ„еҢ…иҫ№з•Ңе’ҢиҒҢиҙЈ
вң“ зҒөжҙ»зҡ„зүҲжң¬з®ЎзҗҶ
вң“ зӢ¬з«Ӣзҡ„жөӢиҜ•е’Ңжһ„е»ә
вң“ workspace иҮӘеҠЁй“ҫжҺҘ
вң“ еҺҹеӯҗжҸҗдәӨ (и·ЁеҢ…жӣҙж”№)
```

## е‘Ҫд»ӨйҖҹжҹҘ

```bash
# жһ„е»әжүҖжңүеҢ…
pnpm build

# жһ„е»әзү№е®ҡеҢ…
pnpm --filter nanomem build

# еҗҜеҠЁејҖеҸ‘жЁЎејҸ
pnpm dev

# жё…зҗҶжүҖжңүжһ„е»әдә§зү©
pnpm clean

# иҝҗиЎҢжөӢиҜ•
pnpm test

# ж·»еҠ дҫқиө–еҲ°зү№е®ҡеҢ…
pnpm --filter nanopencil add lodash

# ж·»еҠ  workspace дҫқиө–
pnpm --filter nanopencil add -D @nanopencil/nanomem@workspace:*

# еҸ‘еёғ
pnpm changeset
pnpm changeset version
pnpm changeset publish
```
