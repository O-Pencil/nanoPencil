# 百度千帆 Coding Plan 配置指南

NanoPencil 支持百度千帆 Coding Plan，可接入 Kimi、DeepSeek、GLM、MiniMax 等模型。

## 获取 API Key

1. 访问 [百度千帆订阅管理页面](https://console.bce.baidu.com/qianfan/resource/subscribe)
2. 选择并购买套餐（Coding Plan Lite 或 Pro）
3. 在订阅管理页面查看并复制专属 API Key

> 注意：Coding Plan 的 API Key 仅用于 Coding Plan 接口，不可用于其他模型。

## 配置方式

### 方式一：首次启动时配置

首次运行 `nanopencil` 时，若未配置任何 Coding Plan API Key，会提示选择：

```
请选择要配置的 Coding Plan：1) 百炼 (Alibaba) 2) 千帆 (Baidu) [1]:
```

选择 `2` 后输入千帆 API Key 即可。

### 方式二：使用 /login 或 /apikey 命令

在交互模式下，使用 `/login` 或 `/apikey` 命令配置 `qianfan-coding` 的 API Key。

## 支持的模型

| 模型 ID | 名称 |
|---------|------|
| kimi-k2.5 | Kimi K2.5 (千帆) |
| deepseek-v3.2 | DeepSeek V3.2 (千帆) |
| glm-5 | GLM-5 (千帆) |
| MiniMax-M2.5 | MiniMax-M2.5 (千帆) |
| glm-4.7 | GLM-4.7 (千帆) |
| MiniMax-M2.1 | MiniMax-M2.1 (千帆) |

## 技术说明

- **Base URL**：`https://qianfan.baidubce.com/v2/coding`
- **API 协议**：兼容 OpenAI chat/completions 接口
- **Provider ID**：`qianfan-coding`
- **auth.json 存储**：`auth.json` 中 `qianfan-coding` 条目存储 API Key

## 套餐额度

- **Lite**：每 5 小时约 1,200 次，每周约 9,000 次，每月约 18,000 次
- **Pro**：每 5 小时约 6,000 次，每周约 45,000 次，每月约 90,000 次

套餐额度仅可在编程工具中使用，禁止用于 API 自动化脚本或批量调用。
