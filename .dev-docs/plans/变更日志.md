# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.10.7] - 2025-03-09

### Added ✨
- **百度千帆与火山方舟 Coding Plan**：添加对百度千帆和火山方舟 Coding Plan 的支持
- **link-world 扩展增强**：支持自动检测并加载 internet-search skill

### Improved 🚀
- **记忆系统提示词优化**：优化 NanoMem 记忆系统提示词，使其更自然类人

## [1.1.0] - 2025-03-03

### Added ✨
- **API Key Management**: Added `/apikey` command to update API keys for current provider without restarting
  - Secure API Key input dialog
  - Automatic provider detection
  - Immediate effect after update
- **Tools Manager Optimization**: Implemented caching mechanism for tool availability checks
  - Reduces filesystem I/O by ~75%
  - Eliminates duplicate download messages on startup
  - Added `clearToolCheckCache()` and `getToolCheckCacheStats()` utility functions

### Improved 🚀
- Better startup performance on Windows (no more repeated "fd not found" / "ripgrep not found" messages)
- Tool path checking now uses in-memory cache for faster lookups

### Fixed 🐛
- Fixed issue where users couldn't update API Key after initial setup
- Fixed repeated tool download prompts on Windows startup

### Documentation 📚
- Added comprehensive API Key command documentation
- Added tools manager optimization documentation
- Added monorepo migration guide

## [1.0.0] - 2025-03-02

### Initial Release
- Basic AI coding agent with read, bash, edit, write tools
- Session management with persistence
- Multi-model support (DashScope Coding Plan, OpenAI, Anthropic, Ollama)
- TUI interface with theme support
- Extension system
- Built-in `/simplify` extension for code refactoring
